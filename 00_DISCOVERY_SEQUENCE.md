# Route U2 Public Discovery Sequence

Date: 2026-06-30

Zenodo record: https://zenodo.org/records/21056615

## Purpose

This sequence is for making the public Route U2 RH proof package discoverable, reviewable, and technically inspectable.

The goal is not to force acceptance by announcement.

The goal is to make the fixed public object easy to inspect.

## Recommended order

1. Add `REVIEWER_START_HERE.md`
2. Add `CLAIM_BOUNDARY.md`
3. Add `ONE_PAGE_TECHNICAL_SUMMARY.md`
4. Add / update a GitHub landing `README.md`
5. Post a restrained public announcement
6. Send individual technical review requests

## Tone

Use this framing:

> I have publicly released a manuscript and Lean-source verification package for a proposed Route U2 proof of the Riemann Hypothesis. Technical comments on the formal boundary, source assumptions, and reproducibility package are welcome.

Avoid this framing at first:

> I proved RH. Please accept it.

## First review target

The first review target should be the boundary and reproducibility structure:

- What is claimed?
- What is formalized?
- What is source-backed?
- What is only a manuscript-level bridge?
- What can be independently rebuilt or checked?
- What exact files are part of the fixed public version?
